# HealthShift Profile Management

## Project Overview
HealthShift is a web application designed to manage profiles for healthcare professionals. The application allows users to manage personal and professional information, upload profile photos, and set availability preferences. Built with modern web technologies, HealthShift leverages Tailwind CSS for styling and provides a clean and user-friendly interface.

## Installation
To install and run the HealthShift application, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/healthshift.git
   ```
2. Navigate to the project directory:
   ```bash
   cd healthshift
   ```
3. Open the `profile.html` file in your preferred web browser to view the application.

## Usage
After opening the `profile.html`, you can:

- Update your personal and professional information.
- Change your profile photo by clicking on the camera icon.
- Choose your availability preferences.
- Save your changes and receive notifications upon successful updates.

## Features
- User-friendly profile management interface.
- Profile photo upload feature with live preview.
- Ability to update personal information, professional details, and availability.
- Notifications for successful updates and actions.

## Dependencies
The following are the key dependencies utilized in this project (as seen in the `profile.html`):

- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/) - An icon toolkit for scalable vector icons.

## Project Structure
The project consists of the following structure:

```
healthshift/
│
├── profile.html  # Main HTML file containing profile management interface
```

### Key Sections of profile.html
- **Header and Sidebar:** Contains navigation links to different sections of the application, such as Available Shifts and My Applications.
- **Profile Section:** Displays user information with options to update personal and professional details.
- **Forms:** Sections for entering personal information, professional details, and availability.
- **Notification Toast:** A dynamic notification system that provides feedback after actions are performed.
